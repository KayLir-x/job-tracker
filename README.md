# Job Tracker

Job Tracker is a full-stack web application for managing job applications, documents, and cover letters in one place. Users can securely log in, create and update applications, upload supporting documents, manage cover letter templates, track application statuses, and view a dashboard with key statistics.

## Tech Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: Prisma + SQLite
- Authentication: JWT
- File Uploads: Multer

## Features

- User authentication with JWT
- Create, edit, delete, and filter job applications
- Track application statuses such as Applied, Interview, Rejected, and Offer
- Upload and manage documents such as CVs, certificates, and portfolios
- Link documents to job applications
- Create, edit, and delete cover letter templates
- Dashboard with application statistics and recent applications
